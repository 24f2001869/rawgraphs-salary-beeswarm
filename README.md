# rawgraphs-salary-beeswarm

Contact: 24f2001869@ds.study.iitm.ac.in

This repository contains a professional Beeswarm Plot visualization created with RAWGraphs.

## Visualization

The beeswarm plot shows employee salary distribution across different departments and experience levels.

- **X-axis**: Salary range (continuous dimension)
- **Y-axis**: Department
- **Color**: Experience level (Junior, Mid, Senior)

## Files

- `chart.png`: RAWGraphs-generated beeswarm plot visualization
